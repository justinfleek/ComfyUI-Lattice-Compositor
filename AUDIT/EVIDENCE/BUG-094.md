# BUG-094: BaseLayer.dispose() Memory Leaks

## Summary
The `dispose()` method doesn't dispose of `effectSourceCanvas` and `motionBlurProcessor`, causing memory leaks when layers are destroyed.

## Severity: LOW

## Location
- File: /ui/src/engine/layers/BaseLayer.ts
- Lines: 1896-1948 (dispose method)
- Undisposed resources: Lines 100, 160

## The Bug
```typescript
// Line 100 - declared but not disposed
protected effectSourceCanvas: HTMLCanvasElement | null = null;

// Line 160 - declared but not disposed
protected motionBlurProcessor: MotionBlurProcessor | null = null;

// Lines 1896-1948 - dispose() method
dispose(): void {
  // Disposes: motionPath, motionPathMarkers, axisGizmo, group contents
  // Does NOT dispose: effectSourceCanvas, motionBlurProcessor

  // ... existing disposal code ...

  // Call subclass-specific disposal
  this.onDispose();
}
// effectSourceCanvas and motionBlurProcessor are NEVER cleaned up!
```

## Proof of Leak
```typescript
const layer = new ImageLayer(layerData);

// Use effects (creates effectSourceCanvas internally)
layer.processEffects(0);

// Enable motion blur (creates motionBlurProcessor)
layer.setMotionBlur(true);
layer.applyMotionBlur(canvas, 0, transform);

// Dispose layer
layer.dispose();

// Result:
// - effectSourceCanvas still in memory (HTMLCanvasElement not removed)
// - motionBlurProcessor still in memory (internal buffers not cleared)
```

## Memory Impact
- effectSourceCanvas: Canvas retains ImageData buffer (width * height * 4 bytes)
- motionBlurProcessor: Contains OffscreenCanvas and frame buffers
- For 1920x1080 layer: ~8MB per undisposed layer
- With frequent layer creation/destruction: Unbounded memory growth

## Related
- 14J: Lifecycle/Cleanup category

## Suggested Fix
```typescript
dispose(): void {
  // Dispose effect source canvas
  if (this.effectSourceCanvas) {
    // Clear canvas to release memory
    this.effectSourceCanvas.width = 0;
    this.effectSourceCanvas.height = 0;
    this.effectSourceCanvas = null;
  }

  // Dispose motion blur processor
  if (this.motionBlurProcessor) {
    this.motionBlurProcessor.clearBuffer();
    this.motionBlurProcessor = null;
  }

  // ... existing disposal code ...
}
```
