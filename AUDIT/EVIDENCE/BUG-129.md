# BUG-129: ControlLayer.setIndicatorSize() No Validation

## Summary
The `setIndicatorSize(size)` method passes size directly to geometry calculations without validation. Invalid values create degenerate indicator geometry.

## Severity: LOW

## Location
- File: /ui/src/engine/layers/ControlLayer.ts
- Lines: 115-126
- Function: setIndicatorSize

## The Bug
```typescript
// Lines 115-126
setIndicatorSize(size: number): void {
  if (size === this.indicatorSize) return;

  this.indicatorSize = size;  // No validation!

  // Recreate indicator with new size
  if (this.indicator) {
    this.group.remove(this.indicator);
    this.disposeIndicator();
    this.createIndicator();  // Uses invalid size
  }
}
```

In `createIndicator()`:
```typescript
const size = this.indicatorSize;
// size / 2 used for line positions
const hPoints = [
  new THREE.Vector3(-size / 2, 0, 0),  // NaN / 2 = NaN
  new THREE.Vector3(size / 2, 0, 0),
];
```

## Proof of Failure
```typescript
const controlLayer = new ControlLayer(layerData, true);

// Attack: size = 0
controlLayer.setIndicatorSize(0);
// Lines drawn from (0,0,0) to (0,0,0) - invisible point

// Attack: size = NaN
controlLayer.setIndicatorSize(NaN);
// Vector3(NaN, 0, 0) - degenerate geometry, invisible

// Attack: size = -100
controlLayer.setIndicatorSize(-100);
// Lines inverted but visible

// Attack: size = Infinity
controlLayer.setIndicatorSize(Infinity);
// Vector3(Infinity, 0, 0) - extreme coordinates
```

## Impact
- Indicator becomes invisible or degenerate
- Only affects editor visualization, not layer functionality
- Low severity: Visual helper only

## Related
- 14A: Numeric inputs - 0, NaN testing

## Suggested Fix
```typescript
setIndicatorSize(size: number): void {
  // Validate size
  if (!Number.isFinite(size) || size <= 0) {
    console.warn('ControlLayer.setIndicatorSize: Invalid size');
    return;
  }

  // Clamp to reasonable range
  size = Math.min(size, 10000);

  if (size === this.indicatorSize) return;

  this.indicatorSize = size;

  // Recreate indicator with new size
  if (this.indicator) {
    this.group.remove(this.indicator);
    this.disposeIndicator();
    this.createIndicator();
  }
}
```
