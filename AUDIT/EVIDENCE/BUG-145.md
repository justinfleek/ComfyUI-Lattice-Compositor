# BUG-145: Object.assign Discards Animation Update

## Summary
onUpdate() uses Object.assign to merge animation data, but when this.modelData.animation is null/undefined, the result is assigned to a temporary object and discarded, losing the update.

## Severity: MEDIUM

## Location
- File: /ui/src/engine/layers/ModelLayer.ts
- Lines: 848-853
- Function: onUpdate()

## Crash Path Trace
```
Function: onUpdate(properties)

Line 848-850:
  if (data.animation !== undefined) {
    Object.assign(this.modelData.animation ?? {}, data.animation);
    if (data.animation.activeClip) {

Step-by-step when this.modelData.animation is null:
  1. this.modelData.animation = null (initial state)
  2. `this.modelData.animation ?? {}` creates NEW empty object {}
  3. Object.assign({}, data.animation) â†’ assigns to temp object
  4. temp object has data.animation properties
  5. temp object is DISCARDED (not assigned anywhere)
  6. this.modelData.animation is STILL null
  7. Line 850: data.animation.activeClip triggers playAnimation()
  8. But animation settings are lost

Result: SILENT NO-OP - animation update not persisted
```

## Code Evidence
```typescript
// Line 847-854
// Handle animation settings
if (data.animation !== undefined) {
  Object.assign(this.modelData.animation ?? {}, data.animation);  // BUG: temp object
  if (data.animation.activeClip) {
    this.playAnimation(data.animation.activeClip);
  }
}
```

## Impact
- Animation settings (speed, loop, autoPlay) are lost
- User tries to configure animation, nothing happens
- playAnimation might work, but other settings are not persisted

## Suggested Fix
```typescript
// Handle animation settings
if (data.animation !== undefined) {
  if (!this.modelData.animation) {
    this.modelData.animation = {
      clips: [],
      time: {
        id: `${this.id}_anim_time`,
        name: 'Animation Time',
        type: 'number' as const,
        value: 0,
        animated: false,
        keyframes: [],
      },
      speed: 1,
      loop: true,
      autoPlay: false,
    };
  }
  Object.assign(this.modelData.animation, data.animation);
  if (data.animation.activeClip) {
    this.playAnimation(data.animation.activeClip);
  }
}
```
