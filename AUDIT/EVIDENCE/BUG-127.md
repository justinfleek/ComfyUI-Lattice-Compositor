# BUG-127: SolidLayer.setDimensions() No Validation

## Summary
The `setDimensions(width, height)` method passes dimensions directly to `THREE.PlaneGeometry` without validation. Invalid values create degenerate or corrupted geometry.

## Severity: LOW

## Location
- File: /ui/src/engine/layers/SolidLayer.ts
- Lines: 177-189
- Function: setDimensions

## The Bug
```typescript
// Lines 177-189
setDimensions(width: number, height: number): void {
  if (width === this.width && height === this.height) {
    return;
  }

  this.width = width;   // No validation!
  this.height = height;

  // Recreate geometry
  this.geometry.dispose();
  this.geometry = new THREE.PlaneGeometry(width, height);  // Degenerate if invalid
  this.mesh.geometry = this.geometry;
}
```

## Proof of Failure
```typescript
const solidLayer = new SolidLayer(layerData);

// Attack: width = 0
solidLayer.setDimensions(0, 1080);
// new THREE.PlaneGeometry(0, 1080) - degenerate mesh, invisible

// Attack: width = NaN
solidLayer.setDimensions(NaN, 1080);
// PlaneGeometry with NaN - corrupted vertex positions

// Attack: width = -1
solidLayer.setDimensions(-1, 1080);
// Negative dimensions - inverted geometry

// Attack: width = Infinity
solidLayer.setDimensions(Infinity, 1080);
// PlaneGeometry(Infinity, 1080) - undefined behavior
```

## Impact
- Degenerate geometry causes solid to be invisible
- NaN corrupts mesh vertex data
- Low severity: Typically called with composition dimensions from trusted source

## Related
- BUG-121: ImageLayer.setDimensions no validation (same pattern)
- BUG-125: ShapeLayer.setSize no validation
- 14A: Numeric inputs - 0, NaN testing

## Suggested Fix
```typescript
setDimensions(width: number, height: number): void {
  // Validate dimensions
  if (!Number.isFinite(width) || !Number.isFinite(height) ||
      width <= 0 || height <= 0) {
    console.warn('SolidLayer.setDimensions: Invalid dimensions');
    return;
  }

  if (width === this.width && height === this.height) {
    return;
  }

  this.width = width;
  this.height = height;

  this.geometry.dispose();
  this.geometry = new THREE.PlaneGeometry(width, height);
  this.mesh.geometry = this.geometry;
}
```
