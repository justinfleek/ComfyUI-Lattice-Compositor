# BUG-110: SplineLayer.setResolution() No Validation

## Summary
The `setResolution(width, height)` method passes dimensions directly to `THREE.Vector2.set()` and `LineMaterial.resolution.set()` without any validation. NaN, zero, or negative values corrupt the resolution used for line width calculations.

## Severity: LOW

## Location
- File: /ui/src/engine/layers/SplineLayer.ts
- Lines: 695-700
- Function: setResolution

## The Bug
```typescript
// Lines 695-700
setResolution(width: number, height: number): void {
  this.resolution.set(width, height);  // No validation!
  if (this.lineMesh) {
    (this.lineMesh.material as LineMaterial).resolution.set(width, height);
  }
}
```

## Proof of Failure
```typescript
const spline = new SplineLayer(layerData);

// Attack 1: NaN resolution
spline.setResolution(NaN, 1080);
// LineMaterial resolution = (NaN, 1080)
// Line width calculations: width * NaN = NaN
// Lines render incorrectly or not at all

// Attack 2: Zero resolution
spline.setResolution(0, 0);
// LineMaterial uses resolution for width calculation
// Division by zero potential in shader
// Lines may become infinitely wide or invisible

// Attack 3: Negative resolution
spline.setResolution(-1920, -1080);
// Negative resolution is meaningless
// Could invert line width calculations

// Attack 4: Infinity
spline.setResolution(Infinity, Infinity);
// Shader calculations with Infinity
// Lines become incorrectly sized
```

## Impact
- Line width calculations corrupted
- Lines render incorrectly or not at all
- Low severity: Usually called from engine resize with valid values
- Connected to LatticeEngine.updateSplineResolutions()

## Related
- BUG-103: LatticeEngine.resize() NaN bypass (feeds this method)
- BUG-105: LatticeEngine.setResolution() NaN bypass
- SYSTEMIC-005: NaN bypass pattern
- 14A: Numeric inputs - NaN, 0, Infinity testing

## Suggested Fix
```typescript
setResolution(width: number, height: number): void {
  // Validate dimensions
  if (!Number.isFinite(width) || !Number.isFinite(height) ||
      width <= 0 || height <= 0) {
    console.warn('SplineLayer.setResolution: Invalid dimensions, keeping current');
    return;
  }

  this.resolution.set(width, height);
  if (this.lineMesh) {
    (this.lineMesh.material as LineMaterial).resolution.set(width, height);
  }
}
```
