# BUG-068: Infinite Loop When burstCount is Infinity

## Summary
triggerBurst() iterates `for (let i = 0; i < emitter.burstCount; i++)`. If burstCount is Infinity, this loop runs forever, freezing the browser.

## Severity: LOW

## Location
- File: /ui/src/engine/particles/GPUParticleSystem.ts
- Lines: 1281, 1288
- Function: triggerBurst

## The Bug
```typescript
// Lines 1277-1293
triggerBurst(emitterId?: string): void {
  if (emitterId) {
    const emitter = this.emitters.get(emitterId);
    if (emitter) {
      for (let i = 0; i < emitter.burstCount; i++) {  // Line 1281 - Infinite if burstCount = Infinity
        this.spawnParticle(emitter);
      }
    }
  } else {
    for (const emitter of this.emitters.values()) {
      if (emitter.burstOnBeat && emitter.enabled) {
        for (let i = 0; i < emitter.burstCount; i++) {  // Line 1288 - Same issue
          this.spawnParticle(emitter);
        }
      }
    }
  }
}
```

## Proof of Failure
```typescript
const system = new GPUParticleSystem({ maxParticles: 1000 });
system.initialize(renderer);

// Add emitter with Infinity burstCount
system.addEmitter({
  id: 'test',
  burstCount: Infinity,
  // ... other config
});

// Trigger burst - infinite loop at line 1281
system.triggerBurst('test');
// Browser tab freezes
```

## Impact
- Browser freeze when burst is triggered
- Requires force-close of browser tab
- User loses unsaved work
- Low severity: Requires specific malicious/accidental config (burstCount = Infinity)

## Suggested Fix
```typescript
triggerBurst(emitterId?: string): void {
  if (emitterId) {
    const emitter = this.emitters.get(emitterId);
    if (emitter) {
      // Validate burstCount
      const count = Number.isFinite(emitter.burstCount) ? emitter.burstCount : 0;
      for (let i = 0; i < count; i++) {
        this.spawnParticle(emitter);
      }
    }
  }
  // ... same fix for line 1288
}
```
