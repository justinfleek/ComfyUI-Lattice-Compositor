# BUG-028: Division by Zero in Fractal Noise with scale=0

## Summary
The `fractalNoiseRenderer()` function uses `params.scale` directly without validation. When `scale=0`, division produces Infinity coordinates, which propagate NaN through the noise functions, resulting in black output instead of noise.

## Severity: LOW

## Location
- File: /ui/src/services/effects/generateRenderer.ts
- Lines: 351, 394
- Function: fractalNoiseRenderer, getOctaveTile

## The Bug
```typescript
// Line 394 - no minimum bound on scale
const scale = params.scale ?? 100;  // scale can be 0!

// Line 351 - division by scale
const sampleX = (x / scale) * frequency;  // x / 0 = Infinity
const sampleY = (y / scale) * frequency;  // y / 0 = Infinity

// Line 354
let noiseValue = smoothNoise(Infinity, Infinity, seed);
// → noise2D uses Math.sin(Infinity * 12.9898 + ...) = NaN
// → returns NaN

// Line 360
tile[y * width + x] = NaN;

// Line 429, 433 - NaN propagates
value += NaN * amplitude;  // NaN
value /= maxValue;         // NaN

// Line 444-446
value = Math.max(0, Math.min(1, NaN));  // NaN (Math.max with NaN returns NaN)
const pixelValue = Math.round(NaN * 255);  // NaN
dst[idx] = NaN;  // Becomes 0 in Uint8ClampedArray
```

## Proof of Failure
```typescript
const params = {
  scale: 0,  // User sets scale to minimum
  complexity: 6
};

fractalNoiseRenderer(input, params);
// Expected: Error or sensible fallback
// Actual: Solid black output (all pixels = 0)
```

## Impact
- Fractal noise with scale=0 produces black image instead of expected noise
- Low severity because scale=0 is edge case beyond documented range (10-10000)
- No crash, but unexpected/corrupted output

## Contrast with Protected Values
Line 395 properly protects complexity:
```typescript
const complexity = Math.max(1, Math.min(20, params.complexity ?? 6));
```

Scale at line 394 lacks equivalent protection:
```typescript
const scale = params.scale ?? 100;  // Should be: Math.max(10, params.scale ?? 100)
```

## Suggested Fix
```typescript
const scale = Math.max(10, params.scale ?? 100);  // Enforce minimum
```
