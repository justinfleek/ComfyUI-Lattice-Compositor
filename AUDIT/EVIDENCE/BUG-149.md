# BUG-149: bankingStrength Infinity Corrupts Camera Rotation

## Summary
applyPathFollowing() calculates bank angle using bankingStrength without validation. When bankingStrength is Infinity, the resulting bank angle is ±Infinity, corrupting the camera rotation.

## Severity: LOW

## Location
- File: /ui/src/engine/layers/CameraLayer.ts
- Lines: 593-594
- Function: applyPathFollowing()

## Crash Path Trace
```
Function: applyPathFollowing(frame, pathFollowing, camera)

Line 574: if (pathFollowing.bankingStrength > 0) {
  - bankingStrength=NaN → NaN > 0 = false → condition skipped (safe by luck)
  - bankingStrength=Infinity → Infinity > 0 = true → enters block

Line 593-594:
  const bankAngle = cross * pathFollowing.bankingStrength * Math.PI / 4;
  this.group.rotateZ(bankAngle);

  - bankingStrength=Infinity:
    - bankAngle = cross * Infinity * Math.PI / 4
    - If cross > 0: bankAngle = +Infinity
    - If cross < 0: bankAngle = -Infinity
    - If cross = 0: bankAngle = NaN (0 * Infinity = NaN)
  - this.group.rotateZ(Infinity) → rotation.z becomes Infinity
  - Three.js rotation corrupted

Result: SILENT CORRUPTION - camera rotation becomes Infinity/NaN
```

## Code Evidence
```typescript
// Line 574-596
if (pathFollowing.bankingStrength > 0) {
  // Calculate curvature from tangent change
  const epsilon = 0.01;
  const tBefore = Math.max(0, t - epsilon);
  const tAfter = Math.min(1, t + epsilon);

  const before = this.splineProvider(pathFollowing.pathLayerId, tBefore, frame);
  const after = this.splineProvider(pathFollowing.pathLayerId, tAfter, frame);

  if (before && after) {
    // Calculate turn direction
    const tangent1 = new THREE.Vector2(before.tangent.x, before.tangent.y).normalize();
    const tangent2 = new THREE.Vector2(after.tangent.x, after.tangent.y).normalize();

    // Cross product for turn direction
    const cross = tangent1.x * tangent2.y - tangent1.y * tangent2.x;

    // Apply banking rotation
    const bankAngle = cross * pathFollowing.bankingStrength * Math.PI / 4;  // No validation
    this.group.rotateZ(bankAngle);  // Infinity corrupts rotation
  }
}
```

## Impact
- Camera rotation becomes invalid (Infinity or NaN)
- Rendering shows garbage or nothing
- Camera view completely broken

## Suggested Fix
```typescript
if (pathFollowing.bankingStrength > 0 && Number.isFinite(pathFollowing.bankingStrength)) {
  // ... calculations ...
  const bankAngle = cross * pathFollowing.bankingStrength * Math.PI / 4;
  if (Number.isFinite(bankAngle)) {
    this.group.rotateZ(bankAngle);
  }
}
```
