# BUG-140: setScale() No Input Validation

## Summary
setScale() passes scale values directly to Three.js without validation. NaN, Infinity, 0, and negative values corrupt the model transform.

## Severity: LOW

## Location
- File: /ui/src/engine/layers/ModelLayer.ts
- Lines: 681-694
- Function: setScale()

## Crash Path Trace
```
Function: setScale(scale: number | { x, y, z })

Line 684-687:
  if (typeof scale === 'number') {
    this.model.scale.setScalar(scale);  // NaN/Infinity passed directly
  } else {
    this.model.scale.set(scale.x, scale.y, scale.z);  // NaN in xyz
  }

  - scale=NaN → model.scale = (NaN, NaN, NaN)
  - scale=0 → model.scale = (0, 0, 0) → model invisible
  - scale=Infinity → model.scale = (Inf, Inf, Inf) → rendering glitch
  - scale=-1 → model.scale = (-1, -1, -1) → model inverted (may be intentional)
  - scale={x:NaN, y:1, z:1} → model.scale = (NaN, 1, 1) → partial corruption

Result: SILENT CORRUPTION - model transform corrupted
```

```typescript
setScale(scale: number | { x: number; y: number; z: number }): void {
  if (!this.model) return;

  if (typeof scale === 'number') {
    this.model.scale.setScalar(scale);  // No validation
  } else {
    this.model.scale.set(scale.x, scale.y, scale.z);  // No validation

  // Update bounding box helper
  if (this.boundingBoxHelper) {
    this.boundingBoxHelper.update();
  }
}
```

## Impact
- Model becomes invisible (scale=0)
- Model transform corrupted with NaN
- Rendering glitches with Infinity
- Bounding box helper also corrupted

## Suggested Fix
```typescript
setScale(scale: number | { x: number; y: number; z: number }): void {
  if (!this.model) return;

  if (typeof scale === 'number') {
    if (!Number.isFinite(scale) || scale <= 0) {
      console.warn(`setScale: invalid scale ${scale}`);
      return;
    }
    this.model.scale.setScalar(scale);
  } else {
    if (!Number.isFinite(scale.x) || !Number.isFinite(scale.y) || !Number.isFinite(scale.z)) {
      console.warn('setScale: invalid scale vector');
      return;
    }
    this.model.scale.set(scale.x, scale.y, scale.z);
  }
  // ...
}
```
