# BUG-171: AudioLayer.getAudioTimeForFrame() Division by fps=0

## Summary
getAudioTimeForFrame() divides frame by fps parameter without validation. When fps=0, division produces Infinity or NaN, corrupting audio synchronization and playback calculations.

## Severity: LOW

## Location
- File: /ui/src/engine/layers/AudioLayer.ts
- Lines: 182-186
- Function: getAudioTimeForFrame()

## Crash Path Trace
```
Function: getAudioTimeForFrame(frame: number, fps: number): number

Line 183:
  const layerTime = frame / fps;  // fps=0 → Infinity or NaN

Line 184:
  const audioTime = (layerTime - this.audioData.startTime) * this.audioData.speed;
  - layerTime=Infinity → audioTime=Infinity (or NaN if startTime/speed are bad)

Line 185:
  return Math.max(0, audioTime);
  - Math.max(0, Infinity) = Infinity
  - Math.max(0, NaN) = NaN (SYSTEMIC-005)

Test cases:
  - fps=0, frame=30: layerTime = 30/0 = Infinity → audioTime = Infinity
  - fps=0, frame=0: layerTime = 0/0 = NaN → audioTime = NaN

Downstream effects:
  - Line 194 isPlayingAtFrame(): calls getAudioTimeForFrame
  - Line 260 startPlayback(): calls getAudioTimeForFrame
  - Line 371 scrubAtFrame(): calls getAudioTimeForFrame
  - All audio sync calculations become invalid

Result: SILENT CORRUPTION (SYSTEMIC-003) - audio sync completely broken
```

## Code Evidence
```typescript
// Lines 182-186
getAudioTimeForFrame(frame: number, fps: number): number {
  const layerTime = frame / fps;  // DIV/0 if fps=0
  const audioTime = (layerTime - this.audioData.startTime) * this.audioData.speed;
  return Math.max(0, audioTime);  // NaN bypasses (SYSTEMIC-005)
}
```

## Impact
- fps=0 produces Infinity/NaN audio time
- isPlayingAtFrame() returns incorrect values
- startPlayback() starts from wrong position
- Audio-video sync completely broken
- SYSTEMIC-003: fps=0 division pattern

## Suggested Fix
```typescript
getAudioTimeForFrame(frame: number, fps: number): number {
  if (!Number.isFinite(fps) || fps <= 0) {
    console.warn(`getAudioTimeForFrame: invalid fps ${fps}`);
    return 0;
  }
  const layerTime = frame / fps;
  const audioTime = (layerTime - this.audioData.startTime) * this.audioData.speed;
  return Math.max(0, audioTime);
}
```
